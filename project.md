# Инструкция Markdown

## Заголовки

Чтобы написать заголовок необходимо поставить знак "#" затем пробел и слово

# один
## два
### три
#### четыре
##### пять

## Списки

Для ненумерованных списков используется знак (+,-,*) перед текстом
+ один
- два
* три

Для нумерованных списков используется знак (цифра и ".") перед текстом

1. один
2. два
3. три
